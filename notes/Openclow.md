# Openclow

## 開啟 / 檢查 OpenClaw（Mac）

### 常用指令

- 檢查狀態：
  ```bash
  openclaw status
  ```

- 檢查 Gateway 服務狀態：
  ```bash
  openclaw gateway status
  ```

- 啟動 Gateway（背景服務 / LaunchAgent）：
  ```bash
  openclaw gateway start
  ```

- 重啟 Gateway：
  ```bash
  openclaw gateway restart
  ```

- 停止 Gateway：
  ```bash
  openclaw gateway stop
  ```

### Control UI

- 本機 Dashboard / Control UI：
  http://127.0.0.1:18789/
